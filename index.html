Домашнее задание

В этом домашнем задании изменять html-код нельзя (кроме 1 пункта), работайте только с css

  1. Создайте css-файл и подключите его к html-странице
  2. Поменяйте шрифт на всей странице: для этого добавьте стиль font-family: Arial, Helvetica, sans-serif; для тега html
  3. Добавьте для всех ячеек и самой таблицы border черного цвета шириной в 1px
  4. Добавьте для тегов td и th стиль padding: 4px
  5. Используя селектор tr:first-child, сделайте текст в первой строке таблицы белого цвета, а фон - зелёного
  6. Также используя псевдокласс :first-child, сделайте выделение колонки id жирным шрифтом.
  7. Наконец, сделайте каждую вторую строку таблицы бледно-зелёного цвета. Для этой задачи удобно использовать селектор tr:nth-child(2n)
  8. Дополнительное задание (необязательное): вместо выделения цветом каждой второй строки таблицы, раскрасьте её в шахматном порядке
